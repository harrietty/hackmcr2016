POST /api/users/:username

Creates a new user and returns it

GET /api/users/:username

Returns specified user

POST /api/games/new

Send req.body.player
Creates a new user and returns it.

POST /api/games/join

Send req.body.playerB and req.body.gameId
Joins playerB to the specified game

POST api/games/update

Send req.body.game and req.body.gameId
Updates the specified game and returns it

POST /api/games/end

Send req.body.gameId
Deletes the specified game and also removes it from the profiles of the players.
